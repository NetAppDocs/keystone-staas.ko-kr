= Keystone 구독의 타임라인을 확인하세요
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
NetApp 콘솔의 Keystone 대시보드는 Keystone 구독에 대한 타임라인 보기를 제공하며 활성화, 수정, 갱신 날짜와 같은 이벤트를 표시합니다. 이 타임라인 보기는 Digital Advisor 에서 사용할 수 없습니다.

구독 일정을 보려면 다음 단계를 따르세요.

.단계
. 콘솔 왼쪽 탐색 메뉴에서 *저장소 > Keystone > 구독*을 선택합니다.
. 구독 기간을 보려면 *구독 번호* 열에서 구독 번호를 선택하세요.
+
*현재 소비량* 탭으로 이동합니다.

. *구독 타임라인* 탭을 선택하세요.
+
image:console-subscription-timeline-graph.png["중요 이벤트와 함께 선 그래프 형식으로 구독 타임라인을 보여주는 스크린샷"]

+
구독 타임라인은 기본적으로 선 그래프로 표시되며, 중요한 이벤트는 아이콘으로 표시됩니다. 같은 날에 여러 유형의 구독 업데이트가 발생하는 경우, 차트에는 개별 아이콘 대신 숫자가 표시됩니다. 숫자 위에 마우스를 올리면 해당 날짜에 발생한 모든 업데이트 유형을 볼 수 있으며, 이벤트 아이콘 위에 마우스를 올리면 날짜 및 확정 용량 변경과 같은 세부 정보를 확인할 수 있습니다. 이러한 이벤트를 표 형식으로 보려면 *View as* 드롭다운 메뉴에서 *Table* 옵션을 선택하세요. 다른 구독 타임라인을 보려면 *Tracking ID* 드롭다운 메뉴에서 선택하거나 검색하세요.

+
image:console-subscription-timeline.png["중요 이벤트와 함께 표 그래프 형식으로 구독 타임라인을 보여주는 스크린샷"]
