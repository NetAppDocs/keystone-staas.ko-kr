= 기간별 데이터 소비량 차트에서 버스트 용량 초과
:allow-uri-read: 


NetApp Console 의 Keystone 대시보드의 새로운 기능 향상:



=== 기간별 데이터 소비량 차트에서 버스트 용량 초과

*소비 추세* 탭의 기간별 데이터 차트에서 이제 버스트 제한을 초과하는 사용량이 강조 표시됩니다. 자세한 내용은 link:./integrations/consumption-tab.html["Keystone 구독의 소비 추세를 확인하세요"]을(를) 참조하십시오.



=== 용량 사용량 예측

*소비 추세* 탭의 *기간별 소비량* 보기에서 *예측* 토글을 사용하여 현재 날짜로부터 향후 120일 동안의 용량 사용량을 예측할 수 있습니다. 자세한 내용은 link:./integrations/consumption-tab.html#view-forecast-data["예측 데이터 보기"]를 참조하십시오.



=== 구독 내보내기의 추적 ID

이제 *구독* 페이지에서 다운로드한 구독 보고서에는 Excel 및 CSV 형식 모두에 대한 성능 서비스 수준 데이터에 *추적 ID*가 포함됩니다.



=== 구독 페이지 열 표시 여부 업데이트

*구독* 페이지에서 *Keystone 버전* 열은 이제 기본적으로 숨겨져 있으며, *연결된 구독* 열은 연결된 구독이 있는 경우에만 표시됩니다.
