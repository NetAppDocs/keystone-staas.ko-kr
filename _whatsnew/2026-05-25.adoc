= 구독 타임라인에서 같은 날짜에 여러 업데이트가 있을 때 표시 방식이 개선되었습니다.

NetApp Console 의 Keystone 대시보드의 새로운 기능 향상:



=== 구독 타임라인에서 같은 날짜에 여러 업데이트가 있을 때 표시 방식이 개선되었습니다.

이제 구독 타임라인 차트에서 여러 유형의 구독 업데이트가 같은 날에 발생할 경우, 개별 아이콘 대신 숫자가 표시됩니다. 숫자 위에 마우스를 올리면 해당 날짜의 모든 업데이트 유형을 확인할 수 있습니다. 다운로드한 보고서에는 각 업데이트에 대한 별도의 행이 포함되어 모든 변경 사항을 완벽하게 기록합니다. 자세한 내용은 link:./integrations/subscription-timeline.html["Keystone 구독의 타임라인을 확인하세요"]를 참조하십시오.
